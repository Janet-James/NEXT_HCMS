Manage Permissions

Step 1: Navigate to the Access Management> Manage Permissions.
Step 2: Select the module name, HTML name, permission name, permission code and the permission description.
Step 3: After entering the above details, Click ”Add”.
Step 4: There is an option to update or remove the module details.
Step 5: The created permission details will be displayed in the tabular column.
