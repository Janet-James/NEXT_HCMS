Manage Reference Items

Manage Reference Item Category

Step 1: Navigate to the Manage Application data> Reference items.
Step 2: Select the reference item category and captures the reference item code, reference item name and the reference item description.
Step 3: After entering the above details, Click ”Add”.
Step 4: There is an option to update or remove the reference item details.
Step 5: The created details will be displayed in the tabular column.
