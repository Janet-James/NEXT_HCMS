Manage Reference Item Category

Step 1: Navigate to the Manage application data> Reference item category.
Step 2: Captures the category name, code and the category description.
Step 3: After entering the above details, Click ”Add”.
Step 4: There is an option to update or remove the reference item category details.
Step 5: The created details will be displayed in the tabular column.
