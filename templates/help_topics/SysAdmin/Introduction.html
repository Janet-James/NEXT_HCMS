Introduction to AppAdmin

AppAdmin

The AppAdmin facilitates the data management, user, access management and also to manage the application data. Data management is a configuration module to set up the master data..

Step 1: Navigate to the Active Users. It displays the number of active users.
Step 2: Navigate to the Unique Roles. It displays the number of unique roles.
Step 3: Navigate to the Permissions. It displays the number of permissions.
Step 4: Navigate to the Modules. It displays the number of modules.
Step 5: Navigate to the Recent Logs. It displays the recent logs with the details of the date and the file size.
Step 6: Navigate to the Quick links. It displays the quick links for the users.
