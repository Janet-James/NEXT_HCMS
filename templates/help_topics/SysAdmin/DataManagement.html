Data Management

This is a configuration module to set up the master data that would be used by the application. These values would appear in the drop down/select fields and serve as a purpose to define the data values and formats.

Step 1: Navigate to the Data management.
Step 2: The data upload screen will display. Select the module name from the drop down menu.
Step 3: Select the class name from the drop down and then upload the script files.
Step 4: After entering the above details, click “Execute”.
Step 5: To remove an incorrect data, click ”Cancel/Clear”.
